一个开源的文档发布工具，专为 MDX 和 OpenAPI 设计

https://github.com/taicode-labs/clarify

基于 MDX 和 OpenAPI，用 TypeScript 写的。走本地优先的 CLI 路线，把 MDX 内容、OpenAPI 规范和 TypeScript 配置拼成静态站点。自带全文搜索、国际化、AI 可读的 llms.txt，还有可定制的 React 渲染器。
