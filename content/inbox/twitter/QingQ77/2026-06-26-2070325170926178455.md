一个可直接部署到 Cloudflare Pages 的部署器面板，帮助用户自动创建或更新 Cloudflare Worker / Pages 服务。

https://github.com/byJoey/cfnew-deployer

用户填上邮箱和 Global API Key，面板自动拉取账户信息、生成随机项目名和 UUID、创建或复用 KV 并绑定。支持 Worker 和 Pages 部署、自定义域名绑定，代码实时从远程仓库拉取。
