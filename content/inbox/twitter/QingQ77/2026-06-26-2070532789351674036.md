通过几行 JavaScript 让任何网页拥有自然语言操控能力，无需浏览器扩展或后端改造

https://github.com/alibaba/page-agent'

阿里巴巴开源的一个浏览器端 GUI Agent 库，纯 TypeScript 写的。在页面里引入一段脚本，用户就能用自然语言指挥网页干活。它通过文本操作 DOM，不需要截图或多模态大模型，支持接自己的 LLM。

提供 npm 包和 CDN 两种用法，还能装 Chrome 扩展跨标签页操作，或者用 MCP Server 让外部 Agent 控制浏览器。适合做 SaaS 的 AI 助手、智能填表、无障碍访问这类场景。
