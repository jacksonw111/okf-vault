一个动静结合的 Nuxt4 博客系统，内容是 Markdown 文件，既可以静态导出也能跑 Node 服务

https://github.com/xurensting/serenade

基于 Nuxt 4 的博客/知识库系统，内容全部用 Markdown 文件管理，不用数据库。

支持两种玩法：静态导出丢到任何托管服务，或者跑 Node 服务提供 SSR、搜索、RSS 等动态功能。有文章、专栏、标签、全文搜索、友链朋友圈、项目展示这些模块，支持 KaTeX 公式、暗黑模式。还能接 OpenAI 自动生成 URL 和封面图，提供 Docker 镜像一键部署。
