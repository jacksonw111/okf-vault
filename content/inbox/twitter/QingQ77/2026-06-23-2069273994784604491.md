在 macOS 上启动本地 HTTP 服务，把系统原生能力（OCR、翻译、人脸识别、二维码、TTS、网页抓取）全暴露成 API 接口，零额外成本。

https://github.com/echosoar/openmac

Swift 写的 Mac 本地 HTTP 服务，把 macOS 系统自带的各种能力包装成了 API。OCR 用 Vision 框架认字，翻译走 macOS 15+ 的 Translation 框架，还能用 WKWebView 做无头浏览器抓网页内容，外加人脸检测、二维码识别、文字转语音。

接口统一 JSON 格式，支持 GET 和 POST，传图片可以用 URL、Base64 或本地路径。完全免费，不依赖任何第三方付费服务。
