给 AI agent 提供观察和操作 iOS 模拟器与 Android 设备屏幕的能力

https://github.com/lycorp-jp/sim-use

一个命令行工具，AI agent 用它就能看手机屏幕内容、点按钮、打字、滑动、按硬件键。它读无障碍树然后输出 token 紧凑的摘要（比原始 JSON 省 16 倍 tokens），用 @N 别名缓存来快速点按。iOS 走模拟器 HID 管道，Android 走 AccessibilityService，接口统一。适合做移动端自动化、Agent 验证开发效果之类的场景。
