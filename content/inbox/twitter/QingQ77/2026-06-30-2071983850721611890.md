把日本数字厅的政府 AI「源内（GENAI）」从 AWS 搬到本地跑，去掉一切云依赖。

 https://github.com/hirokawaguchi/open-genai

Open GENAI 是个非官方项目，让日本数字厅的「源内（GENAI）」能脱离 AWS，完全跑在本地。

用 Keycloak 替代 Cognito 做认证，本地 FastAPI 替代 Lambda/Bedrock，SQLite 替代 DynamoDB，Qdrant 替代 OpenSearch，faster-whisper 替代 Transcribe，Stable Diffusion 替代 Bedrock 图像生成。

Docker Compose 一键启动，LLM 后端可以用 Ollama、vLLM、LM Studio 等任意 OpenAI 兼容接口。

内置 RAG、语音转文字、图像生成、团队管理、AI 应用注册、Dify 工作流集成等功能，全栈开源，macOS Apple Silicon 和 Linux + NVIDIA GPU 都能跑。
