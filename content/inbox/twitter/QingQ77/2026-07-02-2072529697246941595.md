一个用 Rust 和 C 编写的开源项目，通过 JIT 编译技术在 macOS 上直接运行 Linux 容器。

https://github.com/ricccrd/dd

它不需要虚拟机、hypervisor 或 Linux 内核，而是将 JIT 当作客户机的"内核"，在用户态翻译容器指令并处理系统调用。

dd 实现了 Docker Engine API，兼容现有 docker CLI，支持 arm64 Linux、x86-64 Linux（通过 jit86 翻译）和 macOS arm64 三种客户机运行时。

性能方面，在 Apple M5 Pro 上测试显示，arm64 容器计算速度与原生 VM 持平甚至更快，x86-64 容器在九成测试项中超越 QEMU 模拟，浮点运算最高快 24 倍。项目提供 GTK4 桌面应用和 CLI 工具，无需 root 权限。
