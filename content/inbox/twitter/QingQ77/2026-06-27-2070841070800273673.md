让 Pi 编程工具能用上 Claude Code 的 Opus/Sonnet/Haiku 模型，还能在其他 provider 中把任务委托给 Claude Code。

https://github.com/elidickinson/pi-claude-bridge

Pi Claude Bridge 是一个 Pi 扩展，通过 Anthropic 官方的 Agent SDK 把 Claude Code 接进了 Pi。
它有两种玩法：
一是当 provider 用，直接在 Pi 里选 Claude Opus/Sonnet/Haiku 模型，工具调用都在 Pi 的界面里跑；
二是当 AskClaude 工具，用别的 provider 时随时让 Pi 去找 Claude Code 帮忙。

流式输出、MCP 工具、会话恢复、思考链都支持，Pro/Max/Team 订阅都行，默认 200K 上下文，Opus Max 还能开 1M。
