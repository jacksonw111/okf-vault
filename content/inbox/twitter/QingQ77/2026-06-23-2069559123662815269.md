一套适配 Claude Code 与 Codex 的 AI 科研技能包，覆盖文献调研到投稿返修全流程，附带可溯源知识库，不编造文献和数据。

https://github.com/Light0305/Light-skills

Light 把科研全流程拆成 28 个互相衔接的 AI 技能，装进 Claude Code 和 Codex 客户端。每个技能带可跑的脚本、可套的模板和真实范例，背后垫着 9 个可溯源的知识库。从找文献、理数据、想创新点，到写论文、画图、排版、投稿、返修，再到软著专利、答辩 PPT、竞赛申报，都有专门的技能接手。
