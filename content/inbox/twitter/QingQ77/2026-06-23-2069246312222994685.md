为 Claude Code 提供完全离线的持久化项目记忆，避免每次新会话都要重新解释上下文。

https://github.com/raiyanyahya/recall

Recall 是 Claude Code 的本地记忆插件。它自动记录会话日志，然后用 TextRank 算法在本地压缩成摘要，解决每次新会话都要从头解释的"冷启动"问题。所有处理都在本机完成，不调用外部模型或 API，既保护隐私也不浪费订阅令牌。
