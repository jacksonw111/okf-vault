把 LLM Agent 作为系统而非模型来审计，提供从静态建模到沙箱动态验证的端到端安全评估框架。

https://github.com/Gach0ng/AgentStalker

AgentStalker 换个角度做 AI Agent 安全：不测模型本身，而是把 Agent 当整套系统来审计。它分四步走——先把源码抽成污点图，再根据污点图合成攻击链，然后在沙箱里重放攻击路径，最后出判定报告。框架支持 Python 和 Rust 的主流 Agent 框架（LangChain、AutoGen、CrewAI 等），内置 13 类 payload 和 10 条多轮攻击链模板。
