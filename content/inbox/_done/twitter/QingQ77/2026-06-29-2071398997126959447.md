在 Windows 与 macOS 上提供本地优先、可自选 AI 服务商的按住说话听写，把整理后的文字直接贴回当前应用，替代 Wispr Flow 一类订阅式听写工具。

https://github.com/MONKE2525E/Verenu

Tauri + Svelte + Rust + SQLite 的开源桌面听写。全局热键按住录音，松开后音频走你选的转写 API，原文再走你选的清理模型（去口头语、标点、片段与格式规则），最后贴回开始录音时前台的应用。密钥进系统钥匙串，历史和配置在本机 SQLite，没有自建服务器和遥测。
