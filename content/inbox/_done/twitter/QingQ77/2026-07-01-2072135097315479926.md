FlareMo — Cloudflare 原生的个人知识管理系统

https://github.com/realchendahuang/FlareMo

一个运行在 Cloudflare Workers 上的个人笔记系统，利用 Cloudflare 免费套餐（D1 数据库 + R2 对象存储）实现零服务器运维。提供 Flomo 风格的时间线笔记体验，支持标签、附件、搜索、归档、回收站、公开分享链接，并兼容 Memos 的 /api/v1 API 子集。
