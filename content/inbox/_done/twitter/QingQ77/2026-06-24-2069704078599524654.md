一款免费开源的本地 AI 桌面应用，无需付费 API 和云服务即可在个人电脑上创建和管理 AI Agent。

https://github.com/RaulMandujano/evano-studio

Evano Studio 是一个本地优先的 Electron + Python 桌面应用，让普通用户无需接触命令行就能创建、运行和管理本地 AI Agent。

它基于 Ollama 运行本地模型（支持 Gemma、Qwen、Llama、Mistral 等），提供 Agent 创建、多轮对话、多 Agent 团队协作、知识库（RAG）、本地图片生成、日程任务和 Discord 集成等功能。
