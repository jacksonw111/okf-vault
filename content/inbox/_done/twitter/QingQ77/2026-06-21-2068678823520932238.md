一个快速、便携的桌面工具，用来浏览和编辑分散在电脑各处的 Markdown 文档。

https://t.co/OKdlc5aIb3

把阅读、编辑和搜索整合到一个界面里。它用分页方式打开文档，左边是文件树，右边是渲染预览，按 Ctrl+E 切换到源码编辑。支持 GitHub 风格的 Markdown 和 Mermaid 图表，文档之间可以用 文档名 互相链接，右侧大纲栏会同步高亮当前位置。
