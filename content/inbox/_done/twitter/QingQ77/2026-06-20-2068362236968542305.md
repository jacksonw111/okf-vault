Node.js 全能工具包：TypeScript 运行器 + 包管理器 + Node 版本管理器

https://t.co/kqd6Sk4KZf

Node.js 生态的工具太分散了：跑 TS 文件要配 tsconfig，装依赖用 npm/pnpm/yarn 三选一，跑脚本用 npm run，临时调个 CLI 用 npx 慢得要命，换 Node 版本又得 nvm。每个工具都有自己的缓存和配置，切换项目时经常出问题。
