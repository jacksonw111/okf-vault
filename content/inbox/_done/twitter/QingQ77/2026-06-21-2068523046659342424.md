用一台 Linux 服务器跑你所有的应用，git push 后 5 秒内自动部署上线，无需构建队列、无需给每个应用单独付费。

https://t.co/Gfok84Wj0g

Single Server 是一套极简的部署方案：一台 Linux 服务器跑通 Cloudflare、Tailscale、Docker、Kamal 和 GitHub，配置好之后每次 git push 都在 5 秒内发布上线。通过交互式安装脚本初始化，用 singleserver add <repo> 添加仓库即可。
