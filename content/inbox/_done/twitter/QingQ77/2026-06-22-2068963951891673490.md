自托管的 SMTP 中继服务，为家庭网络或小团队提供统一的多租户邮件发送网关，避免每台设备单独配邮箱账号。

https://github.com/toinbox/simplerelay

FastAPI + Postfix + PostgreSQL + Docker 一套搞定。多租户设计，每个用户能加自己的上游 SMTP 提供商，自动检测设置、验 SPF/DKIM/DMARC。每个发件人必须绑 IP 白名单才能用，不然拒绝连接，可选 SMTP AUTH 二次认证。
