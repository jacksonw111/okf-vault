把科研论文、图注、开题材料等研究内容，编译成面向不同 AI 绘图模型的结构化绘图提示词，按领域 x 图类型 x 模型特性自动路由。

https://github.com/BAIKEMARK/happy-figure-skill

这是个 Claude Code / Codex 用的科研绘图 prompt 生成技能。它不画图，而是先读懂科研内容，再根据研究领域、图类型和具体模型，生成能复制、能校对、能控制的 AI 绘图提示词。

覆盖 CS/ML、材料化学、生物医学、地球科学等领域，支持流程图、机制图、图形摘要等 7 种图类型，适配 Nano Banana、Qwen Image、GPT Image 等模型。
