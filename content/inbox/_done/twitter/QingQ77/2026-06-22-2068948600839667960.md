一个面向小团队的轻量级自托管项目管理工具，包含看板任务、客户请求队列和内置 MCP 服务器，可直接部署在 Cloudflare Workers 上。

https://github.com/danielsyauqi/Seeder

部署在 Cloudflare Workers（D1 + R2）或者单台 Node VM 上。功能包括看板任务管理（阶段、标签、优先级、指派、截止日期）、独立的客户请求队列（新建→审核→转化→关闭）、公开客户看板、日常计划表和活动日志。
