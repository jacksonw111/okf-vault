基于 Incus 的 LXC / KVM NAT VPS 销售、交付与管理面板。

https://github.com/qwer-xyz/incudal

一套面向 Incus 的 VPS 销售管理系统，能自动创建和交付 LXC/KVM 实例，支持 NAT 网络。前后端分离，Vue 3 + Fastify，宿主机 Agent 用 Go 写，Docker Compose 一键部署。

除了实例生命周期管理，还集成了用户后台、管理后台、套餐镜像管理、余额充值、VIP 等级和工单系统。
