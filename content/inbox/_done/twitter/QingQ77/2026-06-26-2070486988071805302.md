汇集 Claude Code 最佳实践、开发工作流与社区资源，帮助开发者从"氛围编码"进阶到"智能体工程"

https://github.com/shanraisshan/claude-code-best-practice

GitHub 上最受欢迎的 Claude Code 资源合集，当前拥有超过 60,000 颗星标。内容覆盖 10 个主流开发工作流（如 Superpowers、gstack、OpenSpec）、5 个技能集合和 5 个智能体集合，每个条目均注明来源（Claude 团队、Boris Cherny、Thariq 等社区专家或独立项目作者）。

仓库本身使用 Claude Code 进行维护，包含完整的 CLAUDE.md 配置示例、CLI 设置指南和 83 条经社区验证的技巧与窍门。
