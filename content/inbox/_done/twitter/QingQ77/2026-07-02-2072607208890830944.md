试试 macOS 上跑 AI coding agent 用的原生终端，能并行开多个会话，session 不丢，还能用 CLI 脚本控制。

https://github.com/vestaterm/Vesta

用 Swift/AppKit 写的 macOS 终端，底层直接调 GhosttyKit（libghostty）的 Metal 渲染引擎，不是 Ghostty 的分支。它直接用你现有的 ~/.config/ghostty/config，不用额外配。

最大特点是 session 持久化：vestad 守护进程在后台 hold 住 PTY 和最近 ~256KB 输出，应用退出再打开，Shell 和内容原样恢复。
