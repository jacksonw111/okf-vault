给 Laravel Zero CLI 命令提供可复用的常用功能 trait，不用每次都重新写表格渲染、错误处理和路径解析。

https://github.com/jeffersongoncalves/laravel-zero-console

这是一个 PHP Composer 包，为 Laravel Zero（及普通 Laravel）的命令行命令提供了三个可复用的 trait。FormatsOutput 负责表格渲染、状态颜色化和日期格式化；HandlesApiErrors 把 API 异常统一转为标准退出码；ResolvesPath 处理路径参数与当前工作目录的解析。PHP ^8.2 要求，通过 Composer 安装即可使用，无需额外配置。
