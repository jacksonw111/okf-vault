在 tmux 中管理多个 Claude Code 会话，快速定位需要处理的会话。

https://github.com/fashton28/mux

mux 是个 tmux 插件，装好后按个快捷键就能看到一个浮动面板，列出你所有正在跑的 Claude Code 会话。

哪个在等你输入、哪个跑完了、哪个还卡着，一目了然，而且按等待时间倒排。
