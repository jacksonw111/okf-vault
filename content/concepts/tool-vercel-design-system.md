# Vercel Design System

Vercel 把自己的设计系统以一份公开的「design.md」风格页面形式挂在 [vercel.com/design.md](https://vercel.com/design.md)，覆盖品牌色、字体（Geist / Geist Mono）、设计 tokens、组件规范等内容。作为「设计系统文档应该长什么样」的一个公开样本。

![](https://pbs.twimg.com/media/HLHLZvcboAA9ucD.jpg)

## 它是什么

- 一份**单页 Markdown 风格**的设计系统总览（区别于常见的 Storybook / 0 站文档）
- 公开可访问，无需登录
- 视觉与 Vercel 官网/产品保持一致（Geist 字体、纯黑 / 纯白对比、克制的留白）

## 为什么看它

- 想做自己产品的 design.md 时**直接抄结构**：tokens → 字体 → 颜色 → 间距 → 组件 → 使用示例
- 验证「单页 Markdown 能不能承载一个完整设计系统」——Vercel 给出的是肯定答案
- 与 [Geist UI](https://vercel.com/font) 等资源可组合使用
