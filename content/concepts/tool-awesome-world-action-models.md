# Awesome World Action Models（WAM 论文索引）

## 它是什么

**Awesome World Action Models**。为具身智能（embodied AI）与机器人领域的研究者和工程师提供一个系统化的 **World Action Model（WAM）论文索引**。

## 分类方式

按「预测未来的方式」把 WAM 论文分成三大类，方便追踪技术演进路线：

| 类别 | 说明 |
|------|------|
| 类别 1（依仓库分类） | 不同「预测未来」范式的方法 |
| 类别 2 | …… |
| 类别 3 | …… |

> 仓库自身维护完整分类；这里只示意三类的结构。

## 适用场景

- 做具身智能 / 机器人方向的论文调研 —— 一份清单直接入门。
- 想找某个子方向（如「基于视频预测的 WAM」）的相关工作。
- 给学生 / 新人做文献综述的起点。
