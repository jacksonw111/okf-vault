# AI 视频广告提示词库（52 条）

## 它是什么

**AI 视频广告提示词库**：52 条针对 AI 视频生成场景（广告 / 营销）打磨过的 prompt，按 **10 个类别**整理，每条 prompt 都带：
- 多段时间轴
- 具体镜头运动
- 保真度约束
- 对应的生成示例图

## 覆盖类别（10 个）

| 类别 | 用途 |
|------|------|
| 产品展示 | 突出产品质感、镜头慢推 |
| UGC 风格 | 模拟用户自拍 / 真实感 |
| 前后对比 | 强调效果差异 |
| 开箱 | 拆封、握持、把玩 |
| 问题 → 解决方案 | 痛点引入 → 产品介入 |
| ……等 10 类 | |

## 适用场景

- 直接 copy 到 **Seedance / Veo / Kling / Runway** 等视频生成模型用。
- 不想手写 prompt，又想要稳定质量的广告素材。
- 团队里非 prompt 工程师成员也能产出统一风格的视频。
