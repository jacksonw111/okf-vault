# CS-Fundamentals

## 它是什么
Krishnagangwal 维护的校招 CS 基础复习仓库，把求职笔试面试所需的核心知识汇总在一处。30+ 个 PDF 笔记 + 题库，覆盖七大核心科目 + 求职全流程资料。

## 为什么用它 / 适合什么场景
- 大三 / 大四准备校招，需要一份「一次性把 CS 基础都过一遍」的清单
- 不想自己到处搜集散落的 PDF / 笔记
- 临近面试需要快速过七大科目高频题

## 关键能力（覆盖科目）
| 科目 | 内容 |
|------|------|
| 计算机网络 | 笔记 + 速查表 + 面试题 |
| 数据库与 SQL | 笔记 + 题库 |
| DSA（数据结构与算法） | 笔记 + 题目集合 |
| OOP（面向对象） | 笔记 + 设计题 |
| 操作系统 | 笔记 + 调度 / 内存 / 并发 |
| 软件工程 | 笔记 + 实践原则 |
| 系统设计 | 笔记 + 经典案例（短链 / 限流 / 分布式） |

## 求职配套资源
| 资源 | 说明 |
|------|------|
| HR 面经 | 行为面试常见问题与参考回答 |
| LeetCode 题集 | 按类别整理的高频题 |
| 求职信模板 | Cover Letter 模板 |
| 招聘公司列表 | 可投递公司清单 |

## 注意事项
- 资源大多来自网络搜集，质量与时效性需要自行判断
- PDF 格式为主，不易全文检索
- 适合「集中冲刺复习」而非「系统学习」
