# GPT Image Skills（32 个开箱即用的 AI 配图 Skills）

## 它是什么

把 **32 个 GPT Image 2 配图 Skill** 打包成可即装即用的合集：每个 Skill 都是一份带独立视觉风格定义 + 参考规范的 prompt 模板，安装后通过 **Codex 的 `imagegen` 接口**直接调用。目标是让没有设计能力的用户也能稳定产出特定风格的配图。

## 覆盖的风格

- 青蓝电路发布会风格
- 漫画说明图
- 工程手稿 / 白板手绘
- PPT 信息图
- Notion 插画
- …… 等 32 个

## 适用场景

- 写技术文章 / 文档 / 博客，每次需要「同一套配色 + 同一套风格」的配图 —— 把对应 Skill 作为 system prompt 注入即可。
- 不会设计 / 不会写 prompt，但想产出特定视觉风格 —— 直接用现成的 Skill。
- 团队统一品牌视觉 —— 选 3–5 个 Skill 当作「企业风格指南」。
