# WeatherMaster

## 它是什么
Android 本地天气应用，Kotlin 写的，UI 风格仿 Google Pixel 天气 App。聚合十几家气象数据源，覆盖实时天气与预报，通过 Crowdin 翻译成 30 多种语言。

## 为什么用它 / 适合什么场景
- 想找一个无广告、聚合多源、本地优先的 Android 天气 App。
- 经常出国或关注多源气象数据（比单一官方源更准）。
- 喜欢 Pixel 风格 UI 但没用 Pixel 手机。

## 关键能力
| 能力 | 说明 |
|------|------|
| 平台 | Android |
| 语言 | Kotlin |
| UI 风格 | 仿 Google Pixel 天气 App |

| 功能 | 实时天气 + 预报 |

| 形态 | 开源本地 App |
