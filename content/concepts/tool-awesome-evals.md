# awesome-evals（BenchFlow 带注释清单）

## 它是什么

BenchFlow 团队用三轮深度挖掘整理出来的「构建与评估 AI Agent」最佳资源清单：

1. **1.16 万篇论文**按引用数排序 → 顶级相关工作一览。
2. **针对性扫网** → 补全博客、播客、演讲等论文之外的素材。
3. **47 场演讲 / 播客的逐字记录** → 把口头分享也纳入知识库。

最终产出：**447+ 条带注释的链接 + 146 篇深度笔记**，分 10 个主题章节。

## 适合什么场景

- 想系统学「agent 评估」这一领域的人——从「为什么需要评估」开始到「安全评估」结束。
- 选 baseline / benchmark / LLM-as-judge 时的参考材料库。
- 想做 agent eval 工具的人：附带的 **PATTERNS.md** 含可直接运行的代码示例。
