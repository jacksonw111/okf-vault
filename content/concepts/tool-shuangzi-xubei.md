# shuangzi-xubei（双子续杯）

## 它是什么

"双子续杯"是一个**iPhone 桌面小组件**，让你在锁屏或主屏上**一眼看清 Claude Code 和 Codex 还剩多少额度**，免去打开电脑 / 浏览器 / 控制台查 quota 的麻烦。

名字"双子"指的是 Claude + Codex 这对 AI 编码双子星，"续杯"暗合"看还剩多少、还能不能继续喝（用）"。

## 关键能力

| 能力 | 说明 |
|------|------|
| iPhone 桌面小组件 | 锁屏 / 主屏均可挂载 |
| 实时额度显示 | 不开电脑即看 quota |
| 多服务 | 同时支持 Claude Code 与 Codex |

## 典型使用场景

- 早上出门前瞄一眼手机，确认今天还有多少额度可用
- 通勤路上收到同事求援："能不能帮我调一下这个 bug？"——一眼判断 Claude Code 还够不够用
- 不在公司电脑前，也能估算 token 余量
