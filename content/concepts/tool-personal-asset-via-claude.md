# AI 托管个人资产的方向

## 一句话

把个人资产（账户余额 / 资产配置 / 现金流）的**汇总、分析、决策**交给 AI agent 跑——这既是新的产品形态机会，也是个人财务「自动化」的极端形态。

![个人资产汇总图](https://pbs.twimg.com/media/HLzPh0dbIAE2MPs.jpg)

## 关键观察

| 维度 | 内容 |
|------|------|
| 输入 | 多个账户（银行卡 / 证券 / 基金 / 数字资产 / 负债）的数据 |
| 输出 | 资产汇总图、配置比例、月度趋势 |
| 处理方 | Claude 这类通用大模型 |
| 用户感知 | 一图看懂整体经济状况 |

## 潜在机会

- **个人 CFO agent**：7×24 看账户、生成周报 / 月报
- **资产再平衡提醒**：偏离目标配置时主动提示
- **现金流预警**：预测未来 3–6 个月现金流缺口
- **税务规划**：在合规前提下提示节税空间

## 潜在风险

- **数据安全**：把所有资产数据集中暴露给 AI 服务是巨大攻击面
- **模型幻觉**：财务决策错了代价高
- **合规与隐私**：跨境数据传输、税务申报等敏感场景
- **过度依赖**：失去对个人财务的「手感」

## 适用场景

- 适合做**辅助 + 视图**，而非全自动执行（转账 / 买卖）
- 适合先把「只读」跑通，再考虑「读写」授权
