# MD→Slides（Markdown 原生幻灯片工具）

## 它是什么

Markdown → 幻灯片工具：你写 Markdown，它自动识别 16 种布局（标题、分栏、代码、公式……），渲染成可全屏演示的幻灯片，并支持导出 PPTX。

## 关键能力

| 能力 | 说明 |
|------|------|
| 布局 | 自动识别 16 种（标题 / 分栏 / 代码 / 公式 ……） |
| 富文本 | KaTeX 数学公式、Mermaid 图表、代码高亮 |
| 主题 | 内置 11 套；可装社区主题；可自写 YAML |
| 演示 | 全屏演示 + 演讲者备注 |
| 导出 | PPTX 格式导出（直接拿去汇报） |

## 适用场景

- 写技术分享 / 课程 / 内部 demo —— 直接在 Markdown 里写，配 Mermaid 图 + KaTeX 公式，导出 PPTX 给同事。
- 不爱用 PowerPoint 但要交付 PPTX —— 写 MD，工具自动转。
- 主题想统一管理 —— 写一套 YAML 主题给整个团队用。
