# WaLinux

## 它是什么
WhatsApp 的**轻量级 Linux 原生桌面客户端**，Tauri 构建（前端 JavaScript，后端 Rust）。相比 Electron 方案更省资源，专供 Linux 桌面使用。

## 为什么用它 / 适合什么场景
- 主力系统是 Linux，WhatsApp 官方不提供 Linux 客户端。
- 想用比 Electron 应用更省内存 / 更轻量的方案。
- 想要原生通知与会话保持等桌面集成能力。

## 关键能力
| 能力 | 说明 |
|------|------|
| 平台 | Linux 桌面 |
| 技术栈 | Tauri 2 + JavaScript 前端 + Rust 后端 |
| 资源占用 | 比 Electron 方案更省 |
| 原生通知 | 支持系统原生通知 |
| 会话保持 | 应用退出再打开仍保留会话 |
| 窗口位置记忆 | 自动记住窗口位置 |
| 下载管理 | 支持文件下载 |
