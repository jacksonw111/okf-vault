# liquid-glass（React 液态玻璃折射组件）

## 它是什么

[`samasante/liquid-glass`](https://github.com/samasante/liquid-glass) 是一个**基于 React 的零依赖镜头组件**，在 Chrome / Safari / Firefox 中实现 **Apple 液态玻璃** 实时 DOM 折射效果。

## 为什么用它 / 适合什么场景

- 想给网页加上 Apple Vision Pro / macOS 26 那套「液态玻璃」视觉风格；
- 想**零依赖** —— 不引入 Three.js / WebGL 复杂库；
- 跨主流浏览器：Chrome / Safari / Firefox 都能跑；
- 想做"光透过玻璃看后面 DOM"的实时刻面 / 折射效果。

## 关键能力

| 能力 | 说明 |
|---|---|
| 实时 DOM 折射 | 透过组件看后面内容，实时变形 |
| 零依赖 | 不引 Three.js / WebGL 库 |
| React 组件 | `<LiquidGlass>` 直接用 |
| 跨浏览器 | Chrome / Safari / Firefox |
| Apple 风 | Vision Pro / macOS 26 液态玻璃风格 |
