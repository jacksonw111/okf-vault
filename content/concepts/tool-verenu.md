# Verenu

## 它是什么
Verenu 是一款开源桌面听写应用，主打「本地优先 + 按住说话 + 整理后粘贴」的工作流：按下全局热键录音，松开后音频走你自选的转写 API，原文再走你自选的清理模型（去口头语、标点、片段与格式规则），最后把成品贴回开始录音时前台的应用。技术栈是 Tauri + Svelte + Rust + SQLite。

![](https://pbs.twimg.com/media/HL4RnlUbIAAhdDV.jpg)

## 为什么用它 / 适合什么场景
- **不想订阅 Wispr Flow 这类闭源听写**：所有数据都留在本机；API Key 走系统钥匙串；历史记录与配置存在本地 SQLite。
- **想换不同的转写 / 整理模型**：转写和清理两步各自接外部 API，可以根据语言、领域、隐私要求挑不同的服务商。
- **跨应用粘贴**：录音时哪个应用在前，整理后文字就贴回哪个应用，无需切换窗口。

## 关键能力
| 能力 | 说明 |
|------|------|
| 全局热键按住录音 | 松开即触发整理 + 粘贴流程 |
| 转写 API 可选 | 任意第三方语音转文字服务 |
| 清理模型可换 | 去口头语、加标点、按格式规则整理 |
| 粘贴到前台应用 | 系统级模拟键入，无需切换窗口 |
| 密钥系统钥匙串 | 不进明文配置，不进聊天记录 |
| 本地 SQLite 存历史 | 无自建服务器、无遥测 |

## 隐私与依赖
- 没有自带服务器，没有遥测
- 装好即用，但要选配并填入自己的转写 / 整理 API Key
- 仍依赖外部 API（音频会上传到所选转写服务），「本地优先」指的是配置与历史而非推理
