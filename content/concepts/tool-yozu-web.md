# Yozu Web（企业差旅 AI 的前端 MVP）

## 它是什么

一个**企业差旅 AI 的前端 MVP**：帮出行者从「我想去」一路推到「决定好了可以走」。仓库里包含落地页、行程录入和一套完整演示流程，作为差旅 AI 助手的最小可演示前端。

## 关键能力

| 能力 | 说明 |
|------|------|
| 落地页 | 抓住「我想出差」这一意图的入口 |
| 行程录入 | 收集时间 / 地点 / 偏好，把模糊想法变成结构化数据 |
| 演示流程 | 一套贯穿「想去 → 决定 → 出发」的剧本，方便对外讲故事 / 录 demo |
| 角色定位 | 前端 MVP —— 真正的 AI 后端不在这里，但前端 mock / 编排好了 |

## 适用场景

- 做「AI 差旅助手 / AI 出行规划」类项目时，作为现成前端起点。
- 想给非技术受众演示「AI 怎么把模糊需求一步步收敛到可执行行程」。
- 寻找一个干净的差旅场景前端骨架做二次改造。
