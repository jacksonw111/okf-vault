# Heartmorrow（HMDSimDev/heartmorrow）

## 它是什么

一个完全本地部署的 AI 约会 + 世界模拟器。用户自己写角色卡、搭世界设定、上传立绘；约会过程中由本地大语言模型实时生成对话，但「修改角色关系 / 存档状态」等关键操作必须在服务端校验后才生效——模型与浏览器都没法偷偷改档。

## 为什么用它 / 适合什么场景

- **追剧式 AI 体验**：把 112 天的日历、七维度关系数值、六款小游戏当作沙盒剧本；
- **数据自托管**：玩家记录、角色档案、剧情分支全部本地存储；
- **可扩展经济系统**：可选股票 / 房产 / 赌场等支线，给剧情增加副目标；
- **本地 LLM 试金石**：拿来对比不同本地模型在「角色扮演一致性」上的表现。

## 关键能力

| 模块 | 说明 |
|------|------|
| 角色与世界 | 用户自建 |
| 实时对话 | 本地 LLM 生成 |
