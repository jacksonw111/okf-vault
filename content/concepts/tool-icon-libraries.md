# Web 图标库精选合集

## 它是什么

一个**Web 图标库精选合集**，聚合了当下网络能找到的几乎所有图标源——涵盖 20 万+ 静态图标、海量动效图标、多种视觉风格与品牌图标，并支持把图标导出为 React、SVG 等可直接用于工程的代码格式。

## 覆盖范围

| 维度 | 内容 |
|------|------|
| 数量 | 200,000+ 图标 |
| 类型 | 静态图标 + 动效图标 |
| 风格 | 线 / 面 / 双色 / 手绘 / 3D / 多色 |
| 品牌 | 主流科技 / SaaS 品牌图标 |
| 导出 | React、SVG，以及更多代码格式 |

## 适用场景

- 前端 / 桌面应用快速挑选图标，无需再到处搜
- 设计 / 演示稿需要大量风格统一的图标
- 需要「品牌 + 功能 + 装饰」三类图标一站搞定
- 想要可由代码直接 import 的 React 组件，而不只是 PNG
