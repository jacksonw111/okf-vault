# AgentSpace（人 + AI 代理协作平台）

## 它是什么

[`AgentSpace`](https://github.com/HKUDS/AgentSpace) 是 HKUDS 开源的**面向团队的人 + AI 代理协作平台**。它把 AI 代理当作**有角色、有归属、有权限的团队成员**——而不是用完即走的临时工具——让人类和代理能在同一个协作空间里以团队形式协同工作。

## 关键能力

| 能力 | 说明 |
|------|------|
| 角色化代理 | 给 AI 代理分配明确角色 |
| 团队归属 | 代理有明确的团队归属感 |
| 权限系统 | 代理受权限控制，不能越权 |
| 统一协作空间 | 人类与代理在同一个空间协作 |
| 团队形式 | 而非「一次性调用」的工具 |

## 适用场景

- 团队想把多个 AI 代理编入日常工作流（如研究员 / 工程师 / 审核员）
- 需要对 AI 代理做权限 / 角色管理，避免越权调用
- 想让 AI 代理长期参与项目而不是临时调用
