# 《线性代数不难》开源书

## 它是什么
GitHub 上一本把线性代数讲得「直观 + 可视化」的开源教材《Linear Algebra Made Easy》，作者 Visualize-ML。全套上下两册，从零基础一路进阶，配合大量几何图、可视化和动画，把向量、矩阵、特征值分解等关键概念讲透、讲明白。配套可运行的 `.ipynb` 笔记本，边看边改边跑，把抽象公式变成看得见的直觉。

![](https://pbs.twimg.com/media/HL4sR29bMAALVNa.png)

## 它适合谁
- 想要系统补齐线性代数、并把它真正用到数据分析 / 机器学习里的开发者与学生
- 偏好「图 + 动画 + 代码」三种感官同时调动的学习者
- 喜欢「打开笔记本就能跑」胜过纯公式推导的工程派

## 核心特色
| 特色 | 说明 |
|------|------|
| 全套上下两册 | 覆盖零基础 → 进阶 |
| 几何图 / 可视化 | 把抽象矩阵运算映射成可看图形 |
| 动画 | 关键变换过程动态演示 |
| 可运行 `.ipynb` | 边看边改边跑 |
| 配套 Python | 数据分析 / ML 实战导向 |
