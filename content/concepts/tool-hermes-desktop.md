# hermes-desktop

## 它是什么
为 [Hermes Agent](#) 提供的原生桌面 GUI 客户端。把命令行里的 Agent 管理流程搬到桌面端，给不熟终端的用户一个图形入口。

## 为什么用它 / 适合什么场景
- 已经在用 Hermes Agent 但不想每次都敲命令行。
- 想可视化看 Agent / 任务 / 资源状态，而非纯文本日志。
- 团队里非技术同事想用 Hermes Agent，需要 GUI。

## 关键能力
| 能力 | 说明 |
|------|------|
| 桌面 GUI | 替代 CLI 管理流程 |
| 目标产品 | Hermes Agent（多 AI 代理协作框架） |
| 形态 | 桌面应用 |
