# Refloow Photo Studio（开源桌面照片编辑器）

## 它是什么

Refloow Photo Studio 是 Refloow 出品的免费开源桌面照片编辑应用，主打「无需专业技能也能用、完全本地运行、零数据收集」。

## 关键能力

| 能力 | 说明 |
|------|------|
| 一键移除背景 | 抠图不依赖云端 AI（具体模型未公开） |
| 裁剪 / 镜像 / 调色 | 基础编辑功能齐全 |
| 图层叠加 | 支持多层叠加出合成图 |
| 撤销 / 重做 | 完整历史记录 |
| 纯本地 | 无账号、无广告、无水印，零数据收集 |
| 技术栈 | Electron + Node.js |

## 适用场景

- 想用「无水印、无账号、不联网」的隐私优先照片编辑工具。
- 简单日常图片处理（社交媒体配图、产品图去背景）。
- 不愿用 Adobe / 美图秀秀等闭源桌面工具的替代需求。
