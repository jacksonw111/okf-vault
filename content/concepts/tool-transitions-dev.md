# transitions.dev

## 它是什么

[`transitions.dev`](https://transitions.dev) 是 **Jakub Antalik** 维护的「网页应用过渡效果（Web Page Transitions）」精选集，特点：

- **可直接复制粘贴**——每个效果都附代码片段。
- **真实项目级**——不是动效 demo，是能直接塞到产品里的过渡。
- **覆盖常见场景**——路由切换、列表进出、模态、共享元素（shared element）转场等。

## 为什么这个资源值得收藏

- 前端项目里「页面切换」和「状态变化」是用户感知最强的视觉细节，**做好了立刻高级感翻倍**。
- 网上 90% 的过渡 demo 用的是 Framer Motion / GSAP，但很多人没意识到还可以纯 CSS + View Transitions API 做。
- transitions.dev 给的是「**带应用场景**」的过渡集合，**不是** API 教程。

## 与 [textmotion.dev](tool-textmotion-dev.md) 的关系

| 资源 | 范围 |
|------|------|
| transitions.dev | 页面级 / 组件级过渡（路由、模态、列表） |
| textmotion.dev | 文本级动效（逐字、逐行、打字机、变形） |

两者互补：transitions.dev 管「动效的**结构**」，textmotion.dev 管「动效的**内容**」。

## 适用人群
