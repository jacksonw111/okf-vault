# Free-TV/IPTV

## 它是什么

Free-TV/IPTV 是一个**全球免费电视频道 M3U 播放列表**合集仓库，把各国公开免费的电视直播流整理成 M3U 格式，导入任何 IPTV 播放器即可收看。

- 形态：M3U 播放列表（标准 IPTV 格式）
- 覆盖：全球各国主流免费电视台
- 接入：导入 VLC / IINA / TiviMate / IPTV Pro 等任意 IPTV 播放器
- 来源：聚合公开直播流，非自建服务器

## 关键能力

| 能力 | 说明 |
|------|------|
| M3U 格式 | 标准 IPTV 格式，几乎所有播放器兼容 |
| 全球覆盖 | 各国主流免费台 |
| 开源 | 仓库 + 社区维护 |
| 即开即用 | 导入即看 |

## 典型使用场景

- 想看国外新闻台（BBC / Al Jazeera / NHK World 等）的直播流
- 临时出国想看家乡电视台
- 自建 IPTV 盒子 / 电视墙
- 应急新闻信源（多源对照）
