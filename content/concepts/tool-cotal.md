# Cotal（多智能体开放协议框架）

## 它是什么

Cotal 是一个多智能体开放协议框架：很多多 Agent 系统被固定成「一个指挥 + 多个工人」的层级拓扑，或者只能一对一聊天。Cotal 试图用一套开放协议把多个 Agent 放进同一空间，让用户自由配置拓扑——对等、经理制、指挥链、混搭都能跑在同一套标准上。

## 关键能力

| 能力 | 说明 |
|------|------|
| 开放协议 | 多 Agent 通信不再绑定私有框架 |
| 拓扑可配 | 同一套标准下支持对等 / 经理制 / 指挥链 / 混搭 |
| 统一空间 | 所有 Agent 在同一「空间」里相互可见、可交互 |
| 打破一对一聊天 | 不再只能一对一通信 |

## 适用场景

- 需要灵活编排多 AI 代理（不限于层级）的项目。
- 想做「Agent 团队」但不想被某个具体框架的拓扑假设锁死。
- 多供应商模型混编（Claude / GPT / Gemini / 本地模型）需要统一通信层。
