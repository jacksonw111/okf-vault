# TorLink（终端种子搜索下载）

## 它是什么

TorLink 是个**零设置**的终端种子搜索 + 下载工具，Node.js 写的，安装好 Node 后一行 `npx torlnk` 就能跑起来。它把 FitGirl、YTS、海盗湾、1337x、Nyaa 等常见种子站聚合在同一个搜索框下，结果统一展示，省去逐站切换。

## 关键能力

| 能力 | 说明 |
|------|------|
| 零配置 | 装好 Node 直接 `npx torlnk` |
| 多源聚合 | 同时搜 FitGirl / YTS / 1337x / Nyaa / 海盗湾 |
| 关键词搜索 | 命令行参数即搜 |
| 磁力链支持 | 直接粘贴磁力链下载 |
| 浏览种子库 | 也可按分类浏览站点库 |

## 使用示例

```bash
npx torlnk "ubuntu 24.04"
npx torlnk --browse movies
```
