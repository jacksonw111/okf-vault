# heroicons-animated（316 个动效 Heroicons）

## 它是什么

heroicons-animated 是个把 Tailwind Labs 出品的 [Heroicons](https://heroicons.com) **逐个加上动效**的合集。共 **316 个**动效图标，每个都保留 Heroicons 原生 outline / solid 两套视觉风格，加上预设的进入 / 悬停 / 点击动效。

由 **Vercel** 提供托管与发布支持。

## 关键能力

| 能力 | 说明 |
|------|------|
| 316 个动效图标 | 覆盖 Heroicons 主流图标 |
| 风格一致 | outline / solid 与原版 Heroicons 视觉对齐 |
| 复制即用 | 导出 React / Vue / SVG 等格式 |
| Vercel 托管 | 官网与示例站部署在 Vercel |
| 与 shadcn 兼容 | 可作为 shadcn 项目图标升级 |

## 适用场景

- 后台 / 工具站想给按钮图标加微动效
- 落地页 hero 区用动效图标引导视线
- 替代静态 Heroicons 而不破坏现有 API
