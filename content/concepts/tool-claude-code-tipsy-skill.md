# Claude Code 微醺创意 Skill

## 它是什么

一个 Claude Code 的 **Skill / Prompt 包**：通过模拟人喝到「微醺」时的轻度认知去抑制状态，让 AI 在创意头脑风暴时绕开平时的思考框框，产出既荒诞又有洞见的想法。不是绕过安全过滤，而是**给「规规矩矩」加点「微醺」的去抑制**，配三关过滤防止跑偏。

## 关键能力

| 能力 | 说明 |
|------|------|
| 强度旋钮 | 0.1（仅微醺，思路更柔）→ 1.0（断片，天马行空） |
| 5 种心情 | 哲学 / 混沌 / 忧郁 / 激进 / 俏皮 —— 决定输出倾向 |
| 8 套出招套路 | 系统化的「微醺创意」prompt 模板库 |
| 三关过滤 | ① 不能空洞；② 要同时好笑 + 有干货；③ 不能是「加个啤酒图标就糊弄」的伪创意 |

## 为什么用它 / 适合什么场景

- 头脑风暴卡壳：让 AI 给出意料之外的种子想法，再人工收敛。
- 品牌命名 / slogan / 视觉概念探索：常规 prompt 出来都太「标准答案」，需要一点野路子的种子。
- 内容创作需要「破圈角度」：从「安全版本」切到「微醺版本」对比着选。

## 适用边界

- 严格行业（医疗 / 法规 / 财务）的「正式文档」生成不宜使用 —— 这里产出的「荒诞」是种子不是成品。
- 强度调高时务必人工把关，三关过滤是兜底，不是承诺。
