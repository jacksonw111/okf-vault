# shadcn themes on 21st.dev

## 它是什么
21st.dev 维护的 shadcn/ui 主题聚合站，由 serafimcloud 团队运营。它把社区贡献的所有 shadcn 主题集中在一处，提供浏览 / 实时预览 / 一键复制为 registry 命令 / 一键 ship 到自己的项目。

## 为什么用它 / 适合什么场景
- 想给 shadcn 项目换主题但不知道从哪儿开始
- 设计师交付一个色板，要快速在 shadcn 主题格式里验证效果
- 想找一个 shadcn 官方 59 主题之外的「现成可抄」主题

## 关键能力
| 能力 | 说明 |
|------|------|
| 全社区主题聚合 | 所有 shadcn 主题集中展示，比单独搜 GitHub 快 |
| 浏览器实时预览 | 不需要本地 clone 直接看到组件 + 排版效果 |
| 一键 ship | 几点击生成 `npx shadcn add` 命令直接进项目 |
| 免费 | 21st.dev 平台完全免费 |

## 与 shadcn 官方主题的关系
shadcn 官方提供 59 个主题（在 [themes](https://ui.shadcn.com/themes) 页），21st.dev 是社区驱动的扩展池，主题数量远多于官方，风格更多元（赛博朋克 / 极简 / 暖色 / 玻璃拟态 / 复古等）。
