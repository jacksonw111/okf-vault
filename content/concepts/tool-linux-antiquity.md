# linux-antiquity

## 它是什么
diinki 写的 Hyprland Linux 桌面主题包，定位「古典艺术 + 神话 + 天文」美学风格。涵盖 Wayland 桌面所需的整套视觉元素：状态栏（Quickshell 写）、终端配色、图标主题、天气小部件。

## 为什么用它 / 适合什么场景
- 用 Hyprland 作为日常 Wayland 合成器
- 想换一个「不像默认 Hyprland」的主题（追求古典 / 神话美学）
- 想要一套开箱即用的多组件主题（不用自己拼 Quickshell bar + 图标 + 终端配色）

## 关键能力
| 能力 | 说明 |
|------|------|
| 完整面板 | Quickshell 写的状态栏 / 启动器 |
| 终端配色 | 适配常见终端（Kitty / Foot / Alacritty） |
| 图标主题 | 整套图标包替换 |
| 天气小部件 | 内置天气显示 |
| Art Nouveau + 古典绘画 | 美学风格独特，区别于常见极客风 |
| Hyprland 集成 | 配套窗口动画 / 规则 |

## 注意事项
- 当前 v0.1 是概念验证版本，代码未优化
- 安装前建议备份现有配置（作者明确建议）
- 需要 Hyprland + Quickshell + 相关字体支持
