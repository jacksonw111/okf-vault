# Sophon（sophon.at）

> 来源：[@vikingmute on X 2026-06-05](https://x.com/vikingmute/status/2062707469277855920)

## 它是什么

[`sophon.at`](https://sophon.at) 是一个**聚合展示「AI 全栈相关信息」**的网站：

- 论文 / 最新模型 / Benchmark / 排行榜；
- 论文可直接在线阅读；
- 带 feed，可订阅追踪最新动态。

## 命名彩蛋

**Sophon** 来自《三体》里的「**智子**（Sophon）」——三体人派来锁死地球基础科学的「质子级智能体」。作者大概想表达「一个帮你看住 AI 世界的智能观察站」。

## 为什么值得知道

- **一站式 AI 信息源**——不用每天刷 X / arxiv / Papers with Code 各处。
- **论文可读**——直接在浏览器里读，不用下 PDF。
- **可订阅**——feed 推送，新东西不会漏。
- **.at 域名**——奥地利 ccTLD，作者可能是欧洲圈。

## 关键能力

| 能力 | 说明 |
|------|------|
| 论文聚合 | 每日新论文，按时间 / 热度排 |
| 模型目录 | 最新模型卡片 + 链接到原始 release |
| Benchmark | 跨任务 / 跨模型的横向对比 |
| 排行榜 | 实时刷新排名 |
| 在线阅读 | 论文直接在浏览器打开 |
| Feed 订阅 | RSS / Atom，新动态不漏 |

## 适用人群

- AI 研究者 / 工程师日常追踪前沿；
- 产品经理 / 投资人在做「值得看的新东西」扫描；
- 写「每周 AI 速递」类内容时直接抓最新条目。
