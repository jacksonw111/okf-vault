# GanCook / 干饭厨子（家庭 NAS 点菜系统）

## 它是什么

[`GanCook`](https://github.com/Gloridust/GanCook)（干饭厨子）是一个**面向家庭场景的 NAS 点菜系统**。它把家庭吃饭这件日常事结构化——家人通过选头像登录，在系统自动开启的饭局中各自点菜、认领做饭任务，餐后互相打分，月底在贡献墙查看下厨记录。

![GanCook 界面](https://pbs.twimg.com/media/HLufWwEa4AA0_Kt.jpg) ![贡献墙](https://pbs.twimg.com/media/HLufYptaYAANlCt.jpg)

## 关键能力

| 能力 | 说明 |
|------|------|
| Docker 一键部署 | 无需任何外部数据库或云服务 |
| 头像登录 | 选头像即可识别家人，无需账号密码 |
| 自动饭局 | 系统自动开启饭局，组织点菜 / 做饭 |
| 点菜 & 认领 | 各自点菜 + 自愿认领做饭任务 |
| 餐后打分 | 吃完饭互相打分 |
| 月度贡献墙 | 每月查看谁下厨最多 |

## 适用场景

- 多口之家想要一份「谁做饭 / 谁点菜」的可视化账本
- 想让家里人轮换下厨而不是总由一个人做
- 家里有 NAS / 群晖 / 极空间，想顺便跑一个轻量家庭 App
- 想要零外部依赖的家庭协作工具，数据完全本地
