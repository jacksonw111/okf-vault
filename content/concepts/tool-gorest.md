# Gorest（2D 动画精灵表生成器，由 Codex 驱动）

## 它是什么

[`Gorest`](https://github.com/NO6KIKO/gorest-2d-animation-spritesheet-generator) 是一个**开源的 2D 游戏动画平台**，思路与传统工具**完全不同**：没有一堆按钮让你点，而是直接告诉 Codex agent 你要什么，它帮你生成精灵表、拼场景、调图层，甚至**改工具本身**。

![Gorest 界面](https://pbs.twimg.com/media/HLuFIVsa8AAhLgx.jpg)

## 关键能力

| 能力 | 说明 |
|------|------|
| 自然语言驱动 | 用 Codex 描述需求，agent 帮你实现 |
| 精灵表生成 | 自动产出精灵表 |
| 场景合成 | 多场景管理 + 拼场景 |
| 图层调整 | 调图层关系与顺序 |
| 元数据编辑 | 精灵 / 场景元数据可视化编辑 |
| 侧滚预览 | 横向滚动预览整条场景 |
| 工具自修改 | Codex 也能改 Gorest 自身 |

## 自带示例

项目自带《惊魂笔记》恐怖题材的示例场景和资产库，方便直接上手感受。

## 适用场景

- 2D 游戏原型阶段，需要快速生成大量动画 / 场景
- 想体验「agent 即工具界面」的新工作流
- 需要把精灵 / 场景 / 元数据统一管理在一个工作台里
