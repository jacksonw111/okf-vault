# GameDesignOS（AI 辅助游戏设计操作系统）

## 它是什么

[`DY-2026/GameDesignOS`](https://github.com/DY-2026/GameDesignOS) 是一套**本地优先的 AI 辅助游戏设计操作系统**。

核心理念：把 AI agent 与游戏设计师的对话会话（乱糟糟的聊天记录）**转化为结构化的项目资产**：

- 决策（decision）
- 证据（evidence）
- 实验（experiment）
- 项目资产（asset）

帮助游戏团队**验证创意、诊断体验、撰写策划案**。

## 为什么用它 / 适合什么场景

- 独立游戏 / 小工作室：没有专职 PM/策划文档化 AI 对话，让 AI 输出沉淀为 GDD（Game Design Document）草料；
- 想把"AI 头脑风暴 → 真实设计文档"的链路打通；
- 对云端 AI 不放心、希望本地优先（设计稿 / 创意不会被上传）。

## 关键能力

| 能力 | 说明 |
|---|---|
| 会话 → 结构化 | AI 聊天记录自动转决策 / 证据 / 实验 |
| 创意验证 | 把 AI 提的想法落到可实验的形态 |
| 体验诊断 | 系统化分析用户反馈 / 测试结果 |
| GDD 撰写 | 自动组织成策划案草稿 |
| 本地优先 | 数据不出本机 |
