# ORGII（Rust + Tauri 多 Agent 协作框架）

## 它是什么

**ORGII** —— 用 **Rust + Tauri** 写的开源多智能体开发框架。

## 核心定位

> 卖点不是「帮写代码」，而是「让多个 AI 以组织方式协作」 —— 把 AI Agent 当成你的**持久化同事**。

类比公司：每个 Agent 是个「员工」，有角色、有职责、有持久状态；它们在 ORGII 框架里协作完成任务。

## 关键能力

| 能力 | 说明 |
|------|------|
| 多 Agent 协同 | 多 AI 像组织一样分工 / 协作 |
| 持久化 | Agent 是「同事」而非一次性脚本 |
| 技术栈 | Rust（性能 + 安全）+ Tauri（桌面壳） |
| 开源 | 仓库 `yorgai/ORGII` |

## 适用场景

- 想搭 **multi-agent team**（PM / dev / QA 各一 Agent）。
- 想要本地优先的多 Agent 桌面工具（Tauri）。
- 想给团队演示「AI 同事化」的工程形态。
