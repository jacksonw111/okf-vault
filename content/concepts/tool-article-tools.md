# article-tools（纯前端 HTML 工具集）

## 它是什么

article-tools 是一个纯前端 HTML 工具集，集合四个写作 / 排版场景的零依赖网页工具：封面生成、二维码生成、Markdown 转微信公众号富文本排版、Markdown 转 X 排版。打开浏览器直接用，不装依赖。

## 关键能力

| 工具 | 能力 |
|------|------|
| 封面生成 | 多配色、装饰风、字体可调、PNG 导出 |
| 二维码生成 | 带 Logo 二维码 |
| Markdown → 微信公众号 | 把 Markdown 转成微信公众号富文本可直接粘贴的格式 |
| Markdown → X | 把 Markdown 转成适合 X / Twitter 排版的格式 |

## 技术特点

- 纯前端 HTML，浏览器打开即用
- 零依赖 / 无构建
- 无需后端、无需注册

## 适用场景

- 公众号 / X 双发内容的「配图 + 排版」一步搞定。
- 想离线 / 本地用基础排版工具，又不想用微信编辑器 / 在线版工具。
- 给文章配封面、带 Logo 二维码引流。
