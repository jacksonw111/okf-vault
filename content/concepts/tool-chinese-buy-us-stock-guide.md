# chinese-buy-us-stock-guide（大陆投资者美股实操指南）

## 它是什么

[`chinese-buy-us-stock-guide`](https://github.com/zgwl/chinese-buy-us-stock-guide) 是**写给国内投资者的美股科普**，共九篇正篇，把开户、税务、合规、选券商、入金、出金从头到尾讲了一遍。整体风格偏实操 / FAQ，目标是让一个中国大陆居民读完就能知道从零到第一笔美股交易需要做什么。

## 覆盖范围

| 主题 | 内容 |
|------|------|
| 开户 | 大陆居民如何开美股账户 |
| 税务 | 美国 / 中国两边的税怎么报 |
| 合规 | 外汇额度 / 申报 / 注意事项 |
| 选券商 | 各家券商对比与适用场景 |
| 入金 | 人民币 → 美金的合规路径 |
| 出金 | 美金 → 人民币的合规路径 |

## 适用场景

- 中国大陆居民第一次接触美股 / 港股，需要一份系统化入门
- 已经在投美股，但某些环节（税务 / 出金）没完全搞懂
- 想给父母 / 家人解释美股基本流程
