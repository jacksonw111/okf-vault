# Astro 7

## 它是什么

Astro 是面向内容驱动站点的 Web 框架，核心范式是「默认零 JS、按需 islands」。第 7 个主版本继续在该定位上深耕——把更多能力下沉到内容层，同时让全栈体验更顺滑。

## 为什么关注

- 一份**总览式主版本更新**：文档面向「想一次性了解整个前端工程栈为什么这么设计」的读者。
- 对内容站（博客、文档、营销页、OKF 知识库这类）保持友好。
- 对 agent 友好：可静态导出、可嵌入大量 Markdown，与 OKF 的「一文一概念」契合。

## 关键看点（来自博客）

| 维度 | 趋势 |
|------|------|
| 渲染 | 静态为主，岛屿按需 hydrate |
| 内容 | 把内容当作一等公民，与 Markdown 深度集成 |
| 全栈 | 越来越像「一站式 Web 应用框架」而非「静态站点生成器」 |
| 适配 | 强调与现代 UI 库 / 后端的解耦 |
