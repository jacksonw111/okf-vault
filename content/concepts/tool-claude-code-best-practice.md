# claude-code-best-practice

## 它是什么

claude-code-best-practice 是 **GitHub 上最受欢迎的 Claude Code 资源合集**（60k+ stars），目标是把 Claude Code 用户从"氛围编码（vibe coding）"带向"智能体工程（agent engineering）"。

内容覆盖：

| 类别 | 数量 | 说明 |
|------|------|------|
| 主流开发工作流 | 10 个 | Superpowers / gstack / OpenSpec 等 |
| 技能集合 | 5 个 | Skill 打包合集 |
| 智能体集合 | 5 个 | 多角色 / 多 Agent 编排合集 |
| CLAUDE.md 配置示例 | 完整 | 仓库本身用 Claude Code 维护 |
| 社区验证技巧 | 83 条 | 经社区实战验证的 tips |

每条目均注明来源（Claude 团队、Boris Cherny、Thariq 等社区专家或独立项目作者）。

## 为什么用它

- 一站式：从"如何开始用"到"高阶工作流"全覆盖
- 社区筛选：60k stars = 大量人用并认可
- 可信溯源：每条都注明作者 / 项目
- 自我演示：仓库本身用 Claude Code 维护，README 即 CLAUDE.md 范例
