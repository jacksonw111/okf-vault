# Mineradio（沉浸式 Windows 音乐播放器）

## 它是什么

[`Mineradio`](https://github.com/XxHuberrr/Mineradio) 是一款**基于 Electron 的 Windows 桌面音乐播放器**，目标是把「听歌」升级为「沉浸式音乐体验」：将天气电台、搜索播放、歌词舞台、粒子视觉和 3D 歌单架整合为统一音乐空间。

## 关键能力

| 能力 | 说明 |
|------|------|
| 天气电台 | 天气联动电台 |
| 搜索播放 | 在线搜索并播放 |
| 歌词舞台 | 视觉化的歌词呈现 |
| 粒子视觉 | 粒子动画背景 |
| 3D 歌单架 | 三维歌单展示 |
| 电影镜头 | 镜头式视觉切换 |
| Electron 桌面 | 跨平台基础 |

## 适用场景

- 想把桌面播放器做成「氛围感」音乐空间，而不是普通列表
- Windows 桌面用户想体验 3D / 粒子 + 歌词舞台的多媒体组合
- 需要天气联动的电台感听歌体验
