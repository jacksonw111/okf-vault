# index.how/to/articulate

## 它是什么

[`index.how/to/articulate`](https://index.how/to/articulate) 是 **Emil Kowalski** 等人推出的**设计 / UI/UX 专业术语词典**——把日常讨论视觉时用得上的「精准词汇」按 12 大类整理，每条术语附：

- 简洁、可操作的解释；
- 配套的 **AI skill**（可被 agent 直接调用）。

> 配套资源：作者之前还出过 [animations vocabulary](https://animations.dev/)（描述动画的专业术语词典）。

## 解决的痛点

- 程序员描述设计时常常词穷：「这个按钮动得有点……怪」；
- 设计师和前端沟通时常常鸡同鸭讲：「这个『弹性』到底是 spring 还是 easeOutBack」；
- AI 生成 UI 时 prompt 写「漂亮」= 通用、写「bouncy entrance with 320ms spring」= 可控。

articulate 把这套词**结构化** + **AI 友好**了。

## 12 大类（vikingmute @x 总结 2026-06）

具体大类名称可在官网查阅，覆盖通常包括：

| 类别 | 例子术语 |
|------|----------|
| 动画 | easing、spring、stagger、bounce、overlap、offset |
| 颜色 | hue、saturation、tint、tone、accent、contrast |
| 排版 | hierarchy、rhythm、leading、tracking、measure |
| 间距 | proximity、whitespace、density、breathing room |
| 布局 | alignment、grid、modular、scale |
| 形状 | geometry、radius、stroke、cap |
| 交互 | affordance、feedback、state、transition |
| 可访问性 | contrast、focus、target size |
| 反馈 | affordance、skeuomorphism、brutalism |
| 节奏 / 时间 | duration、choreography、cadence、rhythm |
| 动效曲线 | linear、easeIn、easeOut、easeInOut、spring、bounce、back |
| 视觉权重 | emphasis、dominance、figure/ground |

## 关键能力

| 能力 | 说明 |
|------|------|
| 12 大类 | 覆盖视觉 / 动效 / 交互讨论全场景 |
| 配套 AI skill | 术语 + skill 一起用，agent 可直接调用 |
| 跨界一致 | 设计师 / 程序员 / AI 用同一套词 |
| 精准 prompt | 写「spring with 320ms duration + 0.8 damping」比写「弹一下」强 10 倍 |
| 学习价值 | 想做「设计感强」的产品，先把词背熟 |
