# Fundamental-Ava

## 它是什么
Python 写的**大规模多智能体模拟**框架。目标场景：跑上千个智能体 + 观察群体行为涌现（如角色分工、社会规范形成）。

## 为什么用它 / 适合什么场景
- 想研究**涌现行为**（emergence）：角色分工、规范形成、共识形成。
- 想用统计方法验证涌现是否「真的显著」（不是巧合）。
- 想在框架里跑 BFT 共识、立法投票等「文明层」实验。
- 学术研究：社会学、经济学、组织行为学的多智能体仿真。

## 关键能力
| 能力 | 说明 |
|------|------|
| 规模 | 支持上千个智能体 |
| Agent 构成 | 每个 Agent 独立记忆 + 信念 + 社交模型 |
| 决策循环 | 感知-思考-行动（perception-think-action loop） |
| 记忆分层 | 三层：情节记忆 / 语义记忆 / 程序记忆 |
| 决策机制 | 目标加权 |
| 文明层组件 | 文化规范 / 立法投票 / BFT 共识协议 |
| 涌现检测 | 基于统计检验，告诉你某个群体行为是否真有显著性 |
| 形态 | Python 开源研究框架 |
