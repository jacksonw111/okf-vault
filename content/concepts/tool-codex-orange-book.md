# codex-orange-book（非官方 Codex 全链路指南）

## 它是什么

一份**非官方**的 Codex 全链路使用指南，把 Codex 从「装上就用」到「接进真实项目」的整个过程拆成五大篇章：

1. **基础认知**——Codex 是什么、能干什么、与 Claude Code 等的差异。
2. **安装配置**——从零跑起来。
3. **核心功能详解**——每个主要能力讲透。
4. **标准工作流**——典型开发场景的最佳实践。
5. **实战案例库**——真实项目里的用法示范。

覆盖的进阶主题：自动化、插件、**Skill**、**MCP**、**Git 工作流**、**云端运行**、**记忆系统**。

## 为什么用它

- 对**完全没用过 Codex** 的新手友好，按篇章顺序读即可。
- 对想把 Codex **接入真实项目** 的开发者提供进阶专题。
- 提供**在线阅读 + PDF 下载**两种阅读方式。
- **持续更新**——Codex 能力演进时文档同步。
