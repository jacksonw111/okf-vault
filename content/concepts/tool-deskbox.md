# DeskBox（WinUI 3 桌面整理工具）

## 它是什么

[`Tianyu199509/DeskBox`](https://github.com/Tianyu199509/DeskBox) 是一个 **Windows 11 桌面整理工具**，用 WinUI 3 写，跑在系统托盘里。它能在桌面上生成「桌面小部件（widget）」来：

- **收集文件** — 把散落各处的文件拖到 widget 集中存放；
- **映射已有文件夹** — 把常用目录「映射」到桌面，秒级直达；
- **快速存文本 / 链接 / 截图** — 系统级剪贴板的临时收件箱。

## 为什么用它 / 适合什么场景

- 桌面文件到处都是、想统一收口的人；
- 想有 macOS 那种「桌面堆栈 / 文件抽屉」体验的 Windows 用户；
- 不喜欢装大而全的桌面管理软件、想要一个轻量托盘小工具的人。

## 关键能力

| 能力 | 说明 |
|---|---|
| 系统托盘常驻 | 轻量、不占任务栏空间 |
| WinUI 3 现代化界面 | 跟 Windows 11 设计语言对齐 |
| 桌面小部件 | 文件收集 / 文件夹映射 |
| 快速收集 | 文本 / 链接 / 截图一键收 |
| 全局快捷键 | 不开主界面也能触发 |
